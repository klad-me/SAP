Детальное описание методов
Конструктор

constructor(String fname, byte mode)
  Создает объект File и открывает файл на чтение или запись. Результат выполнения операции открытия файла возвращается методом ok()

Параметры:
  fname - имя файла
mode - режим доступа к файлу: File.READ или File.WRITE
 Метод ok

byte ok()
  Проверить статус открытия файла.

Возвращаемое значение:
  true, если файл открыт успешно, или false - в случае ошибки
Метод seek

void seek(int pos)
  Перейти к указанной позиции в файле
Метод available

int available()
  Получить количество доступных байт для чтения.

Возвращаемое значение:
  количество доступных байт для чтения (только для режима чтения)
Метод readBytes

int readBytes(byte[] buf, int start, int count)
  Читает данные из файла в массив buf.

Параметры:
  buf - массив, в который будут помещены прочитанные данные
start - начальный индекс массива для заполнения
count - количество байт для чтения

Возвращаемое значение:
  количество фактически прочитанных байт, или -1 в случае ошибки ввода/вывода
Метод writeBytes

int writeBytes(byte[] buf, int start, int count)
  Записывает данные в файл из массива buf.

Параметры:
  buf - массив, из которого будут взяты данные для записи
start - начальный индекс массива для записи
count - количество байт для записи

Возвращаемое значение:
  количество фактически записанных байт, или -1 в случае ошибки ввода/вывода
Метод close

void close()
  Закрывает файл. При этом сбрасываются все буфера. Дальнейшие операции над файлом будут невозможны.
