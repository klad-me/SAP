Класс
class DataStream
  Класс DataStream представляет из себя оболочку над интерфейсом Stream. Данный класс позволяем использовать данные с приведением к конкретному типу (в отличии от интерфейса Stream, который оперирует лишь потоками байт).
Конструктор

Создает объект DataStream из указанного потока.

 Метод print

void print(String str)
  Записывает строку в поток. Запись производится в 8-битном формате в кодировке, используемой в языке. Обычно это cp1251. Никакие данные о размере или завершающий символ не записываются.

Параметры:
  str - строка для записи
