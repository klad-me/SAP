Класс
class DateTime implements Compareable
  Класс DateTime предназначен для хранения даты и времени в распакованном формате, а также для различных преобразований даты и времени.
Для хранения даты и времени используются два типа: DateTime (распакованная дата/время) и int (unixtime - количество секунд с 00:00:00 1 янв 1970).
DateTime следует использовать, когда необходимо знать отдельные части даты/времени, а unixtime - в случае, если необходимо сравнить дату/время или вычислить промежуток времени.
