Детальное описание методов
Конструктор

constructor()
  Создает неподключенный сокет.
Метод connect

byte connect(byte[] addr, int port)
  Подключается к TCP-порту. Данная операция блокирует выполняющий поток до успешного подключения или ошибки.

Параметры:
  addr - IP-адрес сервера
port - TCP-порт, к которому необходимо подключиться

Возвращаемое значение:
  true, если подключение завершилось успешно, или false - в случае ошибки
Метод available

int available()
  Возвращает количество доступных для чтения байт.

Возвращаемое значение:
  количество доступных для чтения байт, или -1 в случае ошибки (разрыва соединения)
Метод readBytes

int readBytes(byte[] buf, int start, int count)
  Читает данные из сокета

Параметры:
  buf - буфер для чтения
start - начальный индекс в буфере для заполнения
count - размер читаемых данных

Возвращаемое значение:
  количество прочитанных байт, или -1 в случае ошибки (разрыва соединения)
Метод writeBytes

int writeBytes(byte[] buf, int start, int count)
  Записывает данные в сокет

Параметры:
  buf - буфер для записи
start - начальный индекс в буфере
count - размер записываемых данных

Возвращаемое значение:
  количество записанных байт, или -1 в случае ошибки (разрыва соединения)
Метод close

void close()
  Закрывает соединение.
