Класс
class CRC16
  Класс CRC16 позволяет расчитывать стандартную контрольную сумму CRC16
Поля
static final short CRC_INIT
Начальное значение для CRC16
static final short CRC_OK
Значение, получаемое функцией CRC16, если в нее передать блок данных и его CRC16.
Методы
static short CRC16(short crc, byte[] data, int offs, int size)
Вычислить CRC16 для указанного блока данных. Функция CRC16 сделана таким образом, что если к вычисляемому блоку добавить значение CRC16, то CRC16 нового блока будет равна CRC_OK.

Метод CRC16

static short CRC16(short crc, byte[] data, int offs, int size)
  Вычислить CRC16 для указанного блока данных. Функция CRC16 сделана таким образом, что если к вычисляемому блоку добавить значение CRC16, то CRC16 нового блока будет равна CRC_OK.

Параметры:
  crc - контрольная сумма прошлого блока (для первого блока следует указывать CRC16.CRC_INIT)
data - массив данных
offs - смещение начала данных в массиве
size - расзмер данных в массиве

Возвращаемое значение:
  вычисленная CRC16
