Конструктор

constructor(String devname, int baud)
  Создает объект SerialPort и открывает COM-порт. Результат открытия COM-порта возвращает функция ok().

Параметры:
  devname - имя устройства (для Windows - COM*, для Unix - /dev/tty*)
baud - требуемая битовая скорость
 Метод ok

byte ok()
  Проверить результат открытия COM-порта.

Возвращаемое значение:
  true, если порт был открыт, false - в случае ошибки
Метод available

int available()
  Возвращает количество доступных для чтения байт.

Возвращаемое значение:
  количество доступных для чтения байт
Метод readBytes

int readBytes(byte[] buf, int start, int count)
  Читает данные из порта

Параметры:
  buf - буфер для чтения
start - начальный индекс в буфере для заполнения
count - размер читаемых данных

Возвращаемое значение:
  количество прочитанных байт, или -1 в случае ошибки
