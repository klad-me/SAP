Класс
interface Compareable
  Интерфейс Compareable предназначен для механизма сравнения двух объектов.
Интерфейс состоит из единственного метода - equals.
Методы
byte equals(Object o)
Сравнить вызываемый объект с указанным объектом.

Метод equals

byte equals(Object o)
  Сравнить вызываемый объект с указанным объектом.

Параметры:
  o - объект, с которым необходимо сравнить вызываемый объект

Возвращаемое значение:
  true, в случае, если объекты равны, или false в остальных случаях
