  Получить количество доступных для чтения байт

Метод readBytes

int readBytes(byte[] buf, int start, int count)
  Прочитать данные

Параметры:
  buf - массив, в который будут помещены прочитанные данные
start - начальный индекс массива для заполнения
count - количество байт для чтения

Возвращаемое значение:
  количество фактически прочитанных байт, или -1 в случае ошибки ввода/вывода
Метод writeBytes

int writeBytes(byte[] buf, int start, int count)
  Записать данные

Параметры:
  buf - массив, из которого будут взяты данные для записи
start - начальный индекс массива для записи
count - количество байт для записи

Возвращаемое значение:
  количество фактически записанных байт, или -1 в случае ошибки ввода/вывода
