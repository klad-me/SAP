Список классов
BinaryConfig Класс реализует менеджер бинарных конфигураций.
Фактически этот класс ничего не считывает и не записывает - просто поддерживает хранилище.
Функции чтения/записи должна реализовывать внешняя оболочка.
Byte Позволяет хранить число типа byte
ByteStream Класс для чтения/записи массива байт как потока
CRC16 Класс CRC16 позволяет расчитывать стандартную контрольную сумму CRC16
Compareable Интерфейс Compareable предназначен для механизма сравнения двух объектов.
Интерфейс состоит из единственного метода - equals.
DataStream Класс DataStream представляет из себя оболочку над интерфейсом Stream. Данный класс позволяем использовать данные с приведением к конкретному типу (в отличии от интерфейса Stream, который оперирует лишь потоками байт).
DateTime Класс DateTime предназначен для хранения даты и времени в распакованном формате, а также для различных преобразований даты и времени.
File Класс File предназначен для чтения и записи файлов.
Float Позволяет хранить число типа float
IPC Предоставляет возможности межпроцессного(межнитевого) взаимодействия.
IP_Address Предоставляет методы для работы с IP-адресами.
Integer Позволяет хранить число типа int
Math Предоставляет доступ к некоторым основным математическим функциям
Number Определяет интерфейс для получения числового значения класса
Pack Класс Pack предназначен для паковки и распаковки базовых типов short, int и float в последовательность байт.
При паковке используется формат little-endian (Intel-формат).
Random Класс Random предоставляет возможность получать псевдо-случайные числа.
Runnable Определяет интерфейс для описания нити выполнения (Thread)
Short Позволяет хранить число типа Short
Socket Предоставляет функции работы по протоколу TCP/IP (только клиент)
Stream Определяет интерфейс к потокам данных (входым и выходным)
String Позволяет хранить строку, состоящую из 8-битных символов. Также, описывает базовые операции над строками.
StringBuffer Предоставляет возможность последовательного составления строки.
System Предоставляет некоторые базовые функции исполнительной системы
Thread Предоставляет возможность создавать потоки (Threads), позволяющие параллельное выполнение нескольких методов.
Timer Реализует таймер, считающий в обратную сторону - с установленного значения до нуля. Единица отсчета таймера - 1 миллисекунда.
Vector Предоставляет возможность создания автоматически расширяемого массива (вектора) объектов.
Хранимые элементы приводятся к типу Object.
