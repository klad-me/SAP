Класс
class SerialPort implements Stream
  Предоставляет доступ к последовательным портам контроллера.
Управление передатчиком RS485 берет на себя исполнительная система. При этом, для пользователя RS485 выглядит как обычный полудуплексный потоковый канал связи.

Метод readString

String readString(int timeout, int max_len)
  Прочитать из порта строку, заканчивающуюся символом '\r' или '\n'. В случае, если строка прочитана раньше таймаута, она будет заканчиваться символом '\n' (даже при обнаружении '\r'). При таймауте строка не будет иметь символа новой строки в конце.

Параметры:
  timeout - таймаут чтения в миллисекундах
max_len - максимальная длина принимаемой строки

Возвращаемое значение:
  принятая строка
