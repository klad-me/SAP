Детальное описание методов
Конструктор

constructor(Modbus mbus, byte addr, Modbus_Slave_CallBack c)
  Создает Modbus-функцию и задает обратный вызов для получения и установки значений регистров.

Параметры:
  mbus - используемый драйвер Modbus
addr - адрес устройства, на который должна откливаться данная Modbus-функция
c - обратный вызов для получения и установки значений регистров
 Метод Poll

byte Poll(int timeout)
  Опросить драйвер Modbus на предмет запроса к Modbus-функции, обработать запрос и отправить ответ.

Параметры:
  bufsize - максимально допустимый размер пакета для приема
timeout - время, которое следует ждать запрос

Возвращаемое значение:
  true, если запрос принят и обработан, или false - в случае таймаута приема запроса
